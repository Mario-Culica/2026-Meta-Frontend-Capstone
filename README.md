# Little Lemon Restaurant Web App

This is a **React** project built as part of the Meta Frontend Capstone.

The web app includes:

- Home page with Hero section and **Reserve a Table** button
- Specials section with dishes and **Online Menu**
- Testimonials section with animated customer reviews
- About section with restaurant information
- Booking form with input validation
- Fully responsive layout
- Styled scrollbar and interactive buttons
